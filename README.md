Neighborhood Map
================

A project for Front-End Web Development Nanodegree from [Udacity][1].

**To run the application:**
Download the repository, extract it and open index.html in your browser to see the working tests.

**Resources:**
- [Google Maps APIs][2]
- [MediaWiki][3]
- [jQuery.ajax()][4]
- [knockoutjs.com][5]
- [bootstrap][6]
